reqT Download

Download reqT.jar 30MB v3.1.x-SNAPSHOT latest stable all versions

How to run reqT:

  1. Download the reqT.jar above and save it in some directory
  2. Open a terminal, navigate to that directory and type:
    java -jar reqT.jar
    The above requires java. To check and install java, see below.

  3. Type this into the console by the reqT> prompt and press enter:

    println("Hello reqT!")

  4. Then try out the "Hello reqT" example here

If reqT does not start:

  • Check if you have Java 1.8.x installed by opening a terminal window (bash, cmd, or powershell etc.) and type this command:
    java -version
    This should print something similar to
    java version "1.8.0_151"

  • If the above step failed then install Java 1.8 from here:

    • Windows/Mac: scroll down to Java SE8
    • linux install openjdk-8
